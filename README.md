# Lista Studenti Hogwarts – Harry Potter API (Next.js)

## Descrizione
L’utente può selezionare una casata, esplorare la lista degli studenti e visualizzare i dettagli cliccando sul nome.  
L’interfaccia utilizza **CSS personalizzato** per rappresentare visivamente le atmosfere di Hogwarts, con animazioni e suoni tematici.  

Il progetto dimostra l’uso combinato di:  
- **React Hooks**  
- **Routing dinamico**  
- **Integrazione con API REST tramite Axios**  
- **Componenti riutilizzabili**

## Funzionalità
- Selezione dinamica della casata
- Visualizzazione della lista degli studenti
- Toggle dei dettagli degli studenti cliccando sul nome
- Stile tematizzato con animazioni e gradienti
- Possibile aggiunta di suoni per migliorare l’esperienza utente

## Stack Tecnologico
- **Next.js (React)** – Framework React per applicazioni web moderne
- **Axios** – Client HTTP per il consumo delle API
- **CSS3** – Con gradienti, transizioni e animazioni personalizzate
- **Harry Potter Public API** – [https://hp-api.onrender.com](https://hp-api.onrender.com)
- **TypeScript** – Tipizzazione dei componenti per maggiore sicurezza e manutenibilità
